# AI-Based Resume Screening and Ranking System

## Overview
This project is a simple AI-based system that helps recruiters analyze and rank job applicants based on how well their resumes match a job description.

The system scans resumes, extracts important keywords, and compares them with job requirements to generate a ranking score.

## Features
- Upload resumes
- Extract resume text
- Match resume content with job description
- Calculate matching score
- Rank applicants automatically

## Technologies Used
- Python
- Scikit-learn
- PDFMiner
- Pandas
